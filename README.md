# DisplayTree

An interactive visualization tool designed to explore and understand tree-based data structures through animation, search, and dynamic rendering.

## Overview

DisplayTree is an educational project built to visualize how different tree structures organize, store, and retrieve information.

The application supports multiple tree types, allowing users to experiment with node insertion, traversal, search operations, and structural differences between implementations.

The project was created as part of my exploration of algorithms, data structures, blockchain concepts, and interactive frontend visualizations.

## Supported Structures

### Binary Tree

Visualizes hierarchical node relationships and fundamental tree traversal concepts.

### Patricia Tree

Explores compressed trie structures commonly used for efficient prefix-based storage and retrieval.

### Merkle Tree

Demonstrates hash-based tree structures used in distributed systems, cryptography, and blockchain networks.

## Features

* Interactive tree generation
* Custom node insertion
* Search visualization
* Animated traversals
* Dynamic tree rendering
* Adjustable animation speed
* Multiple tree structures
* Educational visualization interface
